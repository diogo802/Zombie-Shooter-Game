# 🧟 Zombie Shooter Game

Um jogo simples feito com JavaScript onde o jogador se move com WASD e atira com o mouse para eliminar zumbis.

## 🎮 Controles

- W A S D → mover personagem
- Mouse → mirar
- Clique esquerdo → atirar

## 🚀 Como rodar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript (Canvas API)
